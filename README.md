TasteMaghribi est une plateforme e-commerce dédiée à la vente de plats marocains authentiques. Développé avec MongoDB (NoSQL) pour la gestion des données, ce site offre une navigation fluide et intuitive, permettant aux utilisateurs de commander leurs plats préférés en quelques clics.

Le projet intègre un système de gestion des utilisateurs et des commandes, ainsi qu’un espace administrateur pour gérer les catégories, les produits et les utilisateurs.
